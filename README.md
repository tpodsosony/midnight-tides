## Portfolio Website
This is a personal website created using Astro to show off my personal coding projects, with the added use of being used as a hobby website

Included is a home page, projects page, blog page, and a links page. 

#### Features to be added:
1. A media review page
2. Website styling page, to show off css for easy replication for others

_________
#### Set-up
1. Use `npm install` to install dependencies
2. `npm run dev` to run a local dev server or `npm run preview` to build locally
